# Demo

A simple demo repository containing example code and documentation.

## Contents

- `hello_daytona.py` - A simple "Hello World" Python script
- `service-api.md` - API documentation for a media recommendation service (媒资推荐可被集成方案接口使用手册)

## Quick Start

Run the Python demo:
```bash
python hello_daytona.py
```

## API Documentation

See [service-api.md](./service-api.md) for detailed API documentation including:
- Material sync interface (物料同步接口)
- User sync interface (用户同步接口)
- Behavior sync interface (行为同步接口)
- Personalized recommendation interface (个性化推荐接口)
- Monitoring metrics collection (监控metric采集接口)
